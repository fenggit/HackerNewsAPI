# HackerNews API 使用流程图

## API 端点总览

根据 [HackerNews API 官方文档](https://github.com/HackerNews/API)，本服务已实现所有 10 个 API 端点：

### ✅ 已实现的端点清单

| 序号 | 端点 | 路径 | 说明 | 返回数据 |
|------|------|------|------|----------|
| 1 | 获取单个 Item | `GET /api/v0/item/{id}` | 获取 story、comment、job、poll 等 | Item 对象 |
| 2 | 获取用户信息 | `GET /api/v0/user/{id}` | 获取用户详细信息 | User 对象 |
| 3 | 获取最大 Item ID | `GET /api/v0/maxitem` | 获取当前最大的 item ID | 数字 |
| 4 | 获取 Top Stories | `GET /api/v0/topstories` | 获取 Top Stories ID 数组（最多 500 个） | 数字数组 |
| 5 | 获取 New Stories | `GET /api/v0/newstories` | 获取 New Stories ID 数组（最多 500 个） | 数字数组 |
| 6 | 获取 Best Stories | `GET /api/v0/beststories` | 获取 Best Stories ID 数组（最多 500 个） | 数字数组 |
| 7 | 获取 Ask Stories | `GET /api/v0/askstories` | 获取 Ask Stories ID 数组（最多 200 个） | 数字数组 |
| 8 | 获取 Show Stories | `GET /api/v0/showstories` | 获取 Show Stories ID 数组（最多 200 个） | 数字数组 |
| 9 | 获取 Job Stories | `GET /api/v0/jobstories` | 获取 Job Stories ID 数组（最多 200 个） | 数字数组 |
| 10 | 获取更新信息 | `GET /api/v0/updates` | 获取最近更新的 items 和 profiles | 更新对象 |

         │

## 典型使用场景

### 场景 1: 构建 HackerNews 首页

```javascript
// 1. 获取 Top Stories ID 列表
const response = await fetch('https://your-domain.com/api/v0/topstories')
const storyIds = await response.json()

// 2. 获取前 10 个 Stories 的详情
const topStories = await Promise.all(
  storyIds.slice(0, 10).map(id => 
    fetch(`https://your-domain.com/api/v0/item/${id}`)
      .then(res => res.json())
  )
)

// 3. 显示 Stories 列表
console.log(topStories)
```

### 场景 2: 获取用户的所有提交

```javascript
// 1. 获取用户信息
const userResponse = await fetch('https://your-domain.com/api/v0/user/jl')
const user = await userResponse.json()

// 2. 获取用户提交的所有 Items
const userItems = await Promise.all(
  user.submitted.slice(0, 20).map(id =>
    fetch(`https://your-domain.com/api/v0/item/${id}`)
      .then(res => res.json())
  )
)

// 3. 显示用户的提交列表
console.log(userItems)
```

### 场景 3: 监控新内容

```javascript
// 定期检查更新
setInterval(async () => {
  const updatesResponse = await fetch('https://your-domain.com/api/v0/updates')
  const updates = await updatesResponse.json()
  
  // 处理新的 Items
  for (const itemId of updates.items) {
    const itemResponse = await fetch(`https://your-domain.com/api/v0/item/${itemId}`)
    const item = await itemResponse.json()
    console.log('New item:', item)
  }
}, 60000) // 每分钟检查一次
```

## 注意事项

1. **无速率限制**: 当前 API 没有速率限制，但建议合理使用
2. **CORS 支持**: 所有端点都支持跨域请求
3. **数据格式**: 所有响应都是 JSON 格式
4. **时间戳**: 时间字段使用 Unix 时间戳（秒）
5. **空值处理**: 某些字段可能为 `null`，需要做好空值检查
